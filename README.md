# student_result_system.py

Student Result System

A beginner-friendly Python project that calculates a student's total marks, average, grade, and pass/fail result using basic Python concepts.

Features
Takes student marks as input
Stores marks using dictionaries
Calculates total and average
Displays grade based on average
Checks pass/fail condition
Beginner-friendly terminal application
Concepts Used
Variables
Dictionaries
Input/Output
Arithmetic Operators
If-Else Conditions
Technologies
Python 3
Learning Purpose

This project was built while learning Python basics and helped improve logical thinking and problem-solving skills.

Future Improvements
Add multiple students
Add loops for repeated usage
Store data in files
Add functions and error handling
